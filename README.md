FreelancersApp is an app build in SWIFT iOS. This app includes a simple way to create an account in Freelancer App. In this app you can login, if you already
have an account. If you don't have it, you can create new account. You can add bio and based on that people contact you to offer work. 
This app contains a simple map view and web view. Includes pod dependency as 
swifty jSon;
alamofire;
realmSwift;
Includes frameworks: realm and uikit.
